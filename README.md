#FACEID
La app para pasar al tab de favoritos si se prueba en simulador es necesario activar en el simulador la opcion Enrolled para simular el face Id se activa de la siguiente manera:
Con el simulador de iphone abierto en la barra de arriba de la computadora se le da a la opcion Features/Face ID/Enrolled. 

Ahora cuando selecciones el tab favoritos de la app te pedira el simulador faceid para poder simular el reconocimiento facial se hace de la siguiente manera:
Con el simulador de iphone abierto en la barra de arriba se le da a la opcion Features/Face ID/Matching Face. 

Listo con eso se simula el FaceID en simulador
